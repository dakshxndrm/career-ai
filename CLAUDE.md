# CLAUDE.md — Career Atlas

Project context for Claude Code. Read this before any task in this repo.

## What this app is
A career-discovery web app for students. Flow: landing → auth → profile →
AI-generated adaptive quiz → AI career report (top matches, strengths, gaps,
roadmap). Progress is saved to Firestore per user.

## Tech stack (keep)
- React 18 + Vite
- react-router-dom v7 for routing
- Firebase Auth + Firestore (`src/firebase.js`, `src/auth.js`, `src/context/AuthContext.jsx`)
- Serverless function in `api/chat.js` for AI calls (run locally with `vercel dev`)

## AI / model
- The quiz and report are generated by a LOCAL open-source model via **Ollama**.
- `api/chat.js` calls `http://localhost:11434/api/chat` with model `llama3.1:8b`.
- It MUST return the shape `{ content: [{ text }] }` so the frontend keeps working.
- Send `format: "json"` to Ollama; keep the existing `.replace(/```json|```/g,"")`
  cleanup as a safety net. Validate parsed output before use (see Tasks).
- Do NOT add a fine-tuned model. Prompt engineering + validation is the approach.

## Design identity (apply to every page)
Concept: "Chart your course" — an atlas / field-guide feel.
- Palette: ink `#16161D` · paper `#FAF8F3` · marigold `#E0922F` (primary accent) ·
  sage `#2F6B57` (secondary) · mist `#E8E4DA` (borders) · muted text `#6B6B73`
- Type: **Fraunces** (display serif, headings) + **Inter** (body/UI).
- Signature element: faint topographic contour lines (see `DashboardPublic.jsx`).
- Buttons: marigold solid for primary, ink-outline for secondary, 12px radius.
- Reuse the look already established in `src/pages/DashboardPublic.jsx`.
- Accessibility floor: visible focus rings, responsive to mobile, honor
  `prefers-reduced-motion`.

## Known issues to fix
- `api/chat.js` has NO auth or rate limiting — anyone can call it. Lock it down.
- No Firestore security rules — database may be world-readable/writable.
- `Auth.jsx` "Skip Login (Demo Mode)" writes a shared `dummy-user` doc anyone can
  overwrite. Gate or remove it for production.
- `ReviewAssessment.jsx` imports the static `src/data/questions.js` (3 questions)
  which does NOT match the AI-generated questions. It's effectively broken.
